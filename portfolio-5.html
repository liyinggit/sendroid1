二手交易平台

该系统是针对日本严格限制二手物品处理的系统，实现二手物品的在线交易流程，可高效处理用户对二手物品的出品和请求，解决在日本不能随意处理旧物品的痛点。

项目功能

web、android、ios跨平台应用 / 出品 / 请求 / 企业店铺 / 跨应用口令分享 / 用户权限 / 会员管理 / 推送 / 即时聊天 / 即时通知 / 商品收藏 / 商品举报 / 商品和请求评论 / 区域推荐。

项目技术及特点

该系统基于springboot + react + react native 框架进行开发。采用 b/s、c/s 结构，使用MVVM前后端分离框架模式。后端主要使用springboot + jwt + security + websocket，实现逻辑处理、文件存取、权限控制、长连接通信等功能；使用mysql作为数据存储库，通过jdbc连接池加上jpa对数据库进行事务操作；前端主要使用react + react-native实现跨平台应用；使用mobx跨页面管理状态；使用react-native-fs、react-native-fast-image、cookie对数据和图片进行三级缓存；使用react-native-router-flux、react-router控制路由跳转切换；使用material-ui、native-base两种UI框架；使用socket.io、fetch分别进行长连接即时通信以及短链接http数据传输；使用fcm进行跨平台推送；使用appium实现自动化测试；兼容ie/chrome/firefox等主流浏览器以及搭载android 4.0、ios9.0以上系统的手机。
