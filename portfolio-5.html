日本二手交易平台

该系统是针对日本严格限制二手物品处理的系统，实现二手物品的在线交易流程，可高效处理用户对二手物品的出品和请求，解决在日本不能随意处理旧物品的痛点。

项目功能

项目围绕会员进行功能划分，会员可划分为功能可细分为买家功能、卖家功能、通用功能。一个会员既可以是买家也可以是买家。买家可以发布商品请求、可以申请企业店铺。卖家可以上架商品、收藏商品、接收区域内商品推荐、对违规商品进行举报。买卖双方可以通过商品的评论区和聊天室进行沟通协商以达成交易。

项目技术

Spring Framework / Web Socket / AWS S3 / React / React Native / FCM / Appium

项目亮点

兼容ie / chrome / firefox等主流浏览器以及搭载android 4.0、ios9.0以上系统的手机。
