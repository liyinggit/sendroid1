全球医疗项目匹配系统

该系统是提升医疗项目与资金对接效率的系统。系统结合AI技术，实现项目提供方与资金提供方智能匹配。运用国际化特点与实时通讯功能，使跨国项目能够在平台上更加地高效对接。

项目功能

功能包括：全文检索精确匹配，实时通讯，系统用户权限控制，短信及邮箱发送功能，文件上传及导出功能，数据处理与统计，网站国际化（i18n）。系统用户登录后，可根据情况选择业务。如果是项目提供方，则可以发布项目，等待资金提供方联系协商；如果是资金方，可以发布要求，等待项目提供方联系。用户可以进行全局搜索，搜索内容包括发布者上传的文件内容。如果有满足的项目，则可联系发布者，进行通过聊天室私下协商。

项目技术

Spring Framework / Luncene / Web Socket / MySql / Redis / jQuery / Bootstrap

项目亮点

通过i18n 实现多国国际化站点，全站深度检索，AI智能匹配项目。
